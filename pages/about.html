Projeto acadêmico

Uma rede feita para aproximar alunos, comunidades e ideias.

O Conecta Fatec é uma rede social acadêmica desenvolvida para fortalecer a interação entre estudantes da Fatec. A plataforma permite compartilhar publicações, participar de comunidades, encontrar colegas, acompanhar perfis e criar conexões dentro do ambiente universitário.

A proposta do projeto é oferecer um espaço simples, organizado e voltado para a rotina dos alunos, reunindo comunicação, colaboração e divulgação de ideias em uma experiência única. Esta reformulação visual moderniza a interface sem alterar a lógica principal já existente no sistema.
