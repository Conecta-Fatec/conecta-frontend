# Conecta Fatec Frontend

Frontend do Conecta Fatec, uma aplicação de rede social acadêmica desenvolvida para conectar estudantes da Fatec em um ambiente digital colaborativo.

A aplicação é responsável por toda a camada de interface e experiência do usuário, consumindo uma API REST para gerenciar dados e interações em tempo real.

## Sobre o projeto

O Conecta Fatec Frontend foi construído com foco em arquitetura desacoplada, organização de código e consistência visual.

A interface foi projetada para simular o comportamento de redes sociais modernas, permitindo navegação fluida entre diferentes contextos da aplicação, como feed, perfis e comunidades.

Toda a comunicação com o sistema é realizada via API, utilizando autenticação baseada em token, o que garante flexibilidade para integração com diferentes ambientes e escalabilidade da aplicação.

## Principais funcionalidades

* autenticação de usuários via API (token)
* feed dinâmico de publicações
* sistema de curtidas e comentários
* visualização e edição de perfis
* sistema de amizades
* navegação e interação em comunidades

## Arquitetura

O projeto segue o modelo frontend desacoplado, onde a interface atua de forma independente do backend, consumindo endpoints da API para renderização dos dados.

Essa abordagem permite maior escalabilidade, manutenção simplificada e possibilidade de integração com diferentes interfaces no futuro.

## Tecnologias utilizadas

* HTML
* CSS
* JavaScript
* Bootstrap / Tailwind (CDN)
